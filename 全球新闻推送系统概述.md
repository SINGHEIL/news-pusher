# 全球新闻推送系统概述

## 🎯 系统目标

构建一个**自动化全球新闻推送系统**，整合**国内一线消息**和**外网一线消息**，为投资者、新闻关注者提供全面的信息覆盖。

## 📊 系统功能

### 1. 国内新闻抓取
- **政治新闻**：人民日报、新华网等
- **财经新闻**：第一财经、财新网、新浪财经等
- **军事新闻**：环球网军事等
- **公考政策**：广东省人社厅、横琴合作区等

### 2. 外网新闻抓取（新增）
- **国际财经**：路透社、彭博社、华尔街日报、金融时报等
- **国际新闻**：BBC、CNN、纽约时报、The Guardian等
- **亚洲媒体**：日经亚洲、南华早报等
- **金融数据**：加密货币、股市指数、外汇汇率

### 3. 智能处理
- **内容提取**：自动提取新闻正文
- **去重过滤**：基于MD5指纹和URL去重
- **时间过滤**：只抓取24小时内的新闻
- **AI分析**：使用DeepSeek API进行内容摘要和评估

### 4. 微信推送
- **定时推送**：每天9:00自动推送
- **消息分段**：自动处理微信2048字节限制
- **格式优化**：中英双语，分类清晰
- **UTF-8支持**：完美支持中文显示

## 🚀 快速开始

### 环境配置
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入微信和API密钥
```

### 运行测试
```bash
# 测试外网抓取功能
python test_international.py

# 运行国内新闻推送
python run_enhanced_push.py

# 运行全球新闻推送
python run_global_push.py
```

### GitHub Actions自动化
系统已配置两个工作流：
1. **全球推送** (`global-news-push.yml`)：每天9:00推送国内外新闻
2. **国内推送** (`news-push.yml`)：原有工作流，仅推送国内新闻

- **长度优化**：适合微信推送的格式
- **编码处理**：UTF-8编码支持

## ⚠️ 注意事项

### 法律合规
1. **遵守robots.txt**：尊重网站的爬虫规则
2. **合理使用**：仅用于个人学习和研究
3. **注明来源**：保留原始链接和出处

### 技术限制
1. **外网访问**：可能需要代理或VPN
2. **请求频率**：控制请求速度，避免被封
3. **内容更新**：新闻源可能改版，需要定期维护

### 数据安全
1. **API密钥**：妥善保管，不要提交到公开仓库
2. **用户数据**：不收集和存储用户个人信息
3. **内容审核**：注意内容合规性
