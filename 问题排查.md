# 微信推送问题排查

## 当前问题

错误信息：`subscription is canceled`（订阅已取消或关注已取消）

## 可能原因

1. **你未关注测试号**
2. **测试号过期**
3. **接口调用超时**

## 解决方案

### 方案一：检查关注状态

1. 访问微信测试号页面：https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login
2. 查看页面下方的 **"用户列表"**
3. 确认你的微信号是否在列表中
4. 如果不在，用微信扫描页面顶部的二维码，关注测试号

### 方案二：使用正式公众号（推荐）

如果测试号有问题，可以使用正式微信公众号：

1. 注册/使用已有的微信公众号
2. 在 `.env` 文件中修改配置：

```env
WECHAT_TEST=False
WECHAT_APPID=你的正式公众号appid
WECHAT_SECRET=你的正式公众号secret
WECHAT_TEMPLATE_ID=创建的模板消息ID
WECHAT_TOUSER=你的OpenID
```

3. 创建模板消息，获取模板ID

### 方案三：使用邮件推送（备用）

如果微信暂时无法使用，可以改用邮件推送。
