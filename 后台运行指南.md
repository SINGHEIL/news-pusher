# 如何让系统每天自动运行

## Windows后台运行方案

### 方案一：开机自启动（推荐）

1. 按 `Win + R`，输入 `shell:startup`，回车
2. 创建 `start.bat` 的快捷方式
3. 将快捷方式复制到打开的文件夹中
4. 重启电脑后，程序会自动启动

### 方案二：任务计划程序

1. 打开"任务计划程序"（搜索"任务计划程序"）
2. 点击"创建基本任务"
3. 名称：公考财会新闻推送
4. 触发器：每天启动
5. 开始时间：08:00（比推送时间早一点）
6. 操作：启动程序
   - 程序：`python.exe`（查找Python安装路径）
   - 参数：`c:/Users/Rihanna/WorkBuddy/Claw/main.py`
   - 起始于：`c:/Users/Rihanna/WorkBuddy/Claw/`

### 方案三：保持程序运行（简单）

直接双击 `start.bat`，保持窗口开启即可。

**注意：**
- 程序必须保持运行才能自动推送
- 关闭程序需要重新打开 `start.bat`
- 推送时间是每天9:00，可以在 `.env` 文件中修改

## 修改推送时间

编辑 `.env` 文件：

```
PUSH_TIME=08:30  # 改为8:30推送
```

## 查看推送历史

打开 `news_pusher.log` 文件，可以查看每次推送的详细记录。

## 停止推送

按 `Ctrl + C` 停止程序，或直接关闭窗口。
