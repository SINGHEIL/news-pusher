# 📝 操作指南

## 第一步：安装依赖包

**双击运行** `install.bat` 文件，等待安装完成。

---

## 第二步：配置微信

1. 打开浏览器，访问：https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login

2. 微信扫码登录

3. 获取三个信息：
   - **appID**（页面顶部）
   - **appsecret**（页面顶部）
   - **你的微信号**（在"用户列表"中找到你的微信号）

4. 用记事本打开 `.env` 文件，填入：

```env
WECHAT_TEST=True
WECHAT_TEST_APPID=wx1234567890abcdef
WECHAT_TEST_SECRET=abcd1234efgh5678ijkl9012mnop3456
WECHAT_TEST_TOUSER=o6_bmjrPTlm6_2sgVt7hMZOPfL2M
PUSH_TIME=09:00
LOG_LEVEL=INFO
```

5. 保存文件

---

## 第三步：测试推送

1. 用记事本打开 `main.py`

2. 找到最后一行，改为：

```python
scheduler.run(once=True)  # 测试用：立即执行一次
```

3. 保存文件

4. **双击运行** `start.bat`

5. 如果配置正确，你的微信会收到测试消息

---

## 第四步：启动定时任务

测试成功后：

1. 用记事本打开 `main.py`

2. 改回最后一行：

```python
scheduler.run()  # 正式运行：每天定时推送
```

3. 保存文件

4. **双击运行** `start.bat`

5. 保持程序窗口打开，每天9:00会自动推送

---

## 常见问题

**Q: 双击install.bat没反应？**
A: 右键点击install.bat，选择"以管理员身份运行"

**Q: 收不到微信消息？**
A: 检查.env文件中的配置是否正确，查看news_pusher.log日志文件

**Q: 程序运行后闪退？**
A: 用命令行运行：`python main.py`，查看错误信息

**Q: 修改推送时间？**
A: 在.env文件中修改 PUSH_TIME=08:30
