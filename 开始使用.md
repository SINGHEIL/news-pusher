# 🚀 开始使用 - 5分钟快速上手

## 📋 检查清单

在开始之前，请确保您已完成以下准备：

### ✅ 必备条件
1. Python 3.8+ 已安装
2. Git 已安装（可选，用于版本控制）
3. 微信账号（用于接收推送）

### ✅ 推荐条件
1. DeepSeek API 密钥（免费获取）
2. GitHub 账号（用于自动化部署）

---

## 🎯 第一步：配置环境（1分钟）

### 1.1 安装依赖
```bash
# 进入项目目录
cd c:\Users\Rihanna\WorkBuddy\Claw

# 安装Python依赖
pip install -r requirements.txt
```

### 1.2 创建配置文件
```bash
# 复制配置文件
cp .env.example .env
```

### 1.3 编辑配置文件
用文本编辑器打开 `.env` 文件，填入以下信息：

```env
# 微信测试号配置（必须）
WECHAT_TEST=True
WECHAT_TEST_APPID=你的appid
WECHAT_TEST_SECRET=你的secret
WECHAT_TEST_TOUSER=你的OpenID

# AI配置（可选，但推荐）
OPENAI_API_KEY=你的DeepSeek API密钥
OPENAI_BASE_URL=https://api.deepseek.com

# 其他配置
PUSH_TIME=09:00
LOG_LEVEL=INFO
```

## 🎯 第二步：获取配置信息（2分钟）

### 2.1 获取微信测试号信息
1. 访问：https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login
2. 用微信扫码登录
3. 记录以下信息：
   - **appID**：页面显示的AppID
   - **appsecret**：页面显示的AppSecret
   - **OpenID**：在"用户列表"中找到你的微信号，点击显示OpenID

### 2.2 获取DeepSeek API密钥（可选）
1. 访问：https://platform.deepseek.com
2. 注册账号并登录
3. 在API Keys页面创建新密钥
4. 复制API密钥备用

## 🎯 第四步：配置自动化（1分钟）

### 4.1 配置GitHub Secrets（如果使用GitHub）
1. 将项目推送到GitHub仓库
2. 进入仓库 Settings → Secrets and variables → Actions
3. 添加以下Secrets：
   - `WECHAT_APPID` = 你的appid
   - `WECHAT_SECRET` = 你的secret
   - `WECHAT_TOUSER` = 你的OpenID
   - `OPENAI_API_KEY` = 你的DeepSeek API密钥

### 4.2 启用工作流
1. 进入仓库 Actions 页面
2. 启用 "全球新闻推送" 工作流
3. 系统将在每天9:00自动运行

## 🔧 常见配置问题

### 问题1：找不到依赖包
**解决**：
```bash
# 重新安装依赖
pip uninstall -r requirements.txt -y
pip install -r requirements.txt
```

### 问题2：微信配置错误
**解决**：
1. 确认OpenID是否正确（不是gh_开头）
2. 确认测试号是否已关注
3. 检查网络连接

### 问题3：外网抓取失败
**解决**：
1. 检查网络是否可以访问外网
2. 可能需要配置代理
3. 查看详细错误日志

## 🎯 高级配置（可选）

### 修改推送时间
编辑 `.env` 文件：
```env
PUSH_TIME=08:30  # 改为早上8:30
```

### 添加代理服务器
编辑 `international_news_fetcher.py`：
```python
self.proxies = {
    'http': 'http://代理地址:端口',
    'https': 'http://代理地址:端口'
}
```

### 自定义新闻源
编辑 `international_news_fetcher.py`，在 `international_sources` 列表中添加新的新闻源。
