# 外网一线消息抓取指南

## 一、功能概述

本系统新增了**外网一线消息抓取功能**，可以抓取全球主流媒体的最新新闻，包括：

### 支持的新闻源：
1. **国际财经媒体**
   - 路透社 (Reuters)
   - 彭博社 (Bloomberg)
   - 华尔街日报 (Wall Street Journal)
   - 金融时报 (Financial Times)
   - CNBC
   - Yahoo Finance
   - MarketWatch

2. **国际综合媒体**
   - BBC News
   - CNN
   - 纽约时报 (New York Times)
   - The Guardian
   - 美联社 (AP News)
   - 半岛电视台 (Al Jazeera)

3. **亚洲媒体**
   - 日经亚洲 (Nikkei Asia)
   - 南华早报 (SCMP)

4. **金融数据**
   - 加密货币价格 (Bitcoin, Ethereum)
   - 全球股市指数
   - 外汇汇率

## 二、系统架构

```
新系统架构：
├── enhanced_news_fetcher.py      # 国内新闻抓取
```

## 七、GitHub Actions自动化

### 更新工作流配置
修改 `.github/workflows/news-push.yml`：

```yaml
name: Global News Push

on:
  schedule:
    - cron: '0 1 * * *'  # 每天北京时间9:00
  workflow_dispatch:      # 支持手动触发

jobs:
  push-news:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Python
      uses: actions/setup-python@v4
      with:
        python-version: '3.10'
    
    - name: Install dependencies
      run: |
        pip install -r requirements.txt
    
    - name: Run global news push
      env:
        WECHAT_TEST_APPID: ${{ secrets.WECHAT_TEST_APPID }}
        WECHAT_TEST_SECRET: ${{ secrets.WECHAT_TEST_SECRET }}
        WECHAT_TEST_TOUSER: ${{ secrets.WECHAT_TEST_TOUSER }}
        OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
      run: |
        python run_global_push.py
```

## 八、常见问题

### Q1：外网抓取失败怎么办？
**A：**
1. 检查网络连接
2. 可能需要配置代理
3. 新闻源网站可能改版，需要更新选择器

### Q2：内容全是英文怎么办？
**A：** 目前系统抓取的是英文原文。后续可以：
1. 集成翻译API（Google Translate, DeepL）
2. 使用AI进行摘要翻译
3. 添加中文媒体外网版（如CGTN, China Daily）

### Q3：抓取速度慢怎么办？
**A：**
1. 减少每个新闻源的抓取数量
2. 增加请求延迟
3. 使用异步抓取（后续优化）

### Q4：如何添加新的外网新闻源？
**A：**
1. 找到目标网站的新闻列表页
2. 使用浏览器开发者工具分析HTML结构
3. 确定合适的CSS选择器
4. 添加到 `international_news_fetcher.py` 的 `international_sources` 列表
