## 系统文件说明

- `main.py` - 主程序
- `config.py` - 配置文件
- `.env` - 环境变量（包含微信配置）
- `start.bat` - 启动脚本
- `news_pusher.log` - 运行日志

## 修改推送时间

编辑 `.env` 文件：

```
PUSH_TIME=08:30  # 改为8:30推送
```

**Q: 程序关闭后怎么办？**
A: 重新运行 start.bat 即可
