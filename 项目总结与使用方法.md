## 🚀 三步快速上手

### 第1步：环境配置
```bash
# 1. 确保在项目根目录
cd c:\Users\Rihanna\WorkBuddy\Claw

# 2. 安装依赖（如果未安装）
pip install -r requirements.txt

# 3. 创建环境配置文件
copy .env.example .env
# 编辑 .env 文件，填入配置信息
```

### 第2步：配置微信
1. **获取测试号信息**：
   - 访问：https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login
   - 扫码登录，获取 `appID` 和 `appsecret`
   - 在"用户列表"中找到你的**OpenID**（o6_开头，不是gh_开头）

2. **配置 .env 文件**：
```env
# 微信测试号配置
WECHAT_TEST=True
WECHAT_TEST_APPID=你的appid
WECHAT_TEST_SECRET=你的secret
WECHAT_TEST_TOUSER=你的OpenID

# 可选：AI摘要配置
OPENAI_API_KEY=你的DeepSeek API密钥
OPENAI_BASE_URL=https://api.deepseek.com
```

### 第3步：运行测试
```bash
# 测试外网抓取功能
python test_international.py

# 如果测试成功，运行完整推送
python run_global_push.py
```

## 🚀 GitHub Actions自动化

### 全球推送工作流
- **文件**：`.github/workflows/global-news-push.yml`
- **时间**：每天北京时间9:00
- **内容**：国内外新闻完整推送

### 配置步骤
1. 在GitHub仓库的 **Settings → Secrets and variables → Actions**
2. 添加以下Secrets：
   - `WECHAT_APPID`
   - `WECHAT_SECRET`
   - `WECHAT_TOUSER`
   - `OPENAI_API_KEY`（可选）

3. 工作流将自动：
   - 每天9:00运行
   - 抓取国内外新闻
   - 推送到微信
   - 保存日志和统计

## 🔍 故障排除

### 常见问题1：外网抓取失败
**解决方法**：
1. 检查网络连接
2. 可能需要配置代理
3. 查看 `global_news_push.log` 日志

### 常见问题2：微信推送失败
**解决方法**：
1. 检查 `.env` 文件配置
2. 确认OpenID是否正确
3. 查看微信测试号是否正常

### 常见问题3：AI分析失败
**解决方法**：
1. 检查API密钥是否有效
2. 确认DeepSeek API服务状态
3. 使用备用分析模式
